# PCT-100 化工控制实训装置

基于 ESP32-C3 的智能环境控制器，适用于化工控制实训教学。

- 🌡️ DS18B20 温度监测 + 自动风扇控制
- ☀️ LDR 光照检测 + 自动 LED 补光
- 💡 双路继电器（LED 灯 / 风扇），支持自动/手动模式
- 📡 MQTT 物联远程监控与控制
- 🖥️ OLED 实时显示 + WS2812 多彩状态指示
- 🔧 串口命令行配置，FreeRTOS 多任务调度
