1. 产品总览

PCT-100 智能环境控制器是基于 ESP32-C3 微控制器设计的嵌入式实训装置，适用于化工控制实训场景。装置集成温度采集（DS18B20）、光照强度检测（LDR）、双通道继电器控制以及 MQTT 远程通信能力，可作为传感器数据采集与执行控制的实训教学平台。

核心亮点

  • 四层软件架构（驱动 → 传输 → 业务 → 编排），模块间单向依赖，代码可维护性高
  • 支持 WiFi 无线联网与 MQTT 远程控制，实现物联网数据交互
  • 0.96 寸 OLED 显示屏实时显示温度、光照、继电器状态
  • WS2812 多彩指示灯提供 6 种优先级动画状态反馈
  • Async WiFi 配网 + 串口命令行调试，无需额外编程器
  • 基于 FreeRTOS 多任务调度，各模块独立运行互不阻塞

2. 系统架构

软件采用严格的四层单向依赖架构，下层不知道上层的存在，确保修改的可预测性。

编排层 · src.ino / scheduler / device
▼
业务层 · controller / mqtt_handler / ws2812_indicator
▼
传输层 · wifi_drv / mqtt_client_drv / serial_cmd
▼
驱动层 · key / relay / ldr / ds18b20 / oled_ui / ws2812_drv

各层之间仅向下依赖，全局状态通过 SystemState 结构体共享

3. 核心功能

🌡️

温度监测

DS18B20 数字温度传感器，精度 ±0.5°C，测量范围 -55°C ~ +125°C。阈值可配置，超限自动启动风扇。

☀️

光照检测

光敏电阻（LDR）模拟采样，阈值可设，低于设定值自动开启 LED 补光照明。

💡

双路继电器

独立控制 LED 灯与风扇两路 220V 设备。支持自动/手动两种工作模式，手动模式支持 00→01→10→11 编码循环。

📡

MQTT 物联

支持远程状态查询与控制。发布 chemctrl/{id}/status，订阅 chemctrl/{id}/command。配置持久化至 NVS。

🖥️

OLED 显示

0.96 寸 128×64 单色 OLED，I2C 接口，实时显示温度、光照、继电器及模式状态。

🎨

WS2812 指示灯

6 级优先级动画指示系统状态：WiFi 连接、MQTT 状态、继电器工作模式。20ms 刷新实现平滑渐变。

4. 硬件规格

主控芯片ESP32-C3 RISC-V 单核处理器，160MHz
温度传感器DS18B20 数字式，1-Wire 接口，精度 ±0.5°C
光照传感器光敏电阻（LDR），模拟电压输出
继电器输出2 路，5V 有源高电平驱动，250V/10A
显示屏0.96" OLED 128×64，I2C（地址 0x3C）
状态指示灯WS2812 RGB LED
按键输入2 个轻触按键（总闸 + 模式/循环）
通信接口UART (115200 baud) / WiFi 2.4GHz IEEE 802.11 b/g/n
电源5V DC（USB 或端子供电）

6. 通信协议与串口命令

MQTT 协议

发布主题chemctrl/{device_id}/status
订阅主题chemctrl/{device_id}/command
状态推送机制事件驱动（状态变化时自动推送 JSON）
支持命令set_relay / set_mode / set_threshold / get_status / reboot

串口命令 (115200 baud)

命令说明
wifi scan / list / connWiFi 扫描与配网
wifi status / forget连接状态查询 / 清除凭据
mqtt status查看 MQTT 连接及配置
mqtt set key=val在线修改 MQTT 配置（持久化 NVS）
mqtt reconnect手动触发 MQTT 重连
set temp/light=val修改控制阈值
set mode=auto/manual切换工作模式
get status查看全部系统参数

7. WS2812 状态指示说明

RGB 指示灯通过 6 级优先级动画反馈系统运行状态，优先级从高到低排列：

1 — WiFi 断连

红 200ms → 绿 200ms → 灭 500ms 循环

2 — MQTT 断连

红 → 渐绿 500ms → 渐灭 → 渐蓝 500ms → 渐灭 循环

3 — 双继电器开启

RGB 交替闪烁 (RGB / 灭 各 50ms)

4 — 仅 LED 开启

R 300ms → 灭 200ms → G 300ms → 灭 200ms → B 300ms → 灭 200ms

5 — 仅风扇开启

R 500ms → G 500ms → B 500ms → 灭 700ms

6 — 全部正常

灭 → 渐红 1s → 渐绿 1s → 渐蓝 1s → 渐灭 1s → 灭 200ms
